# Snake game for GitHub — files + README This document contains everything you need to add a playable **Snake** game to your repo and make it available from your **GitHub README** via a link (README cannot run scripts directly). It includes: * `index.html` — a single-file HTML+CSS+JS Snake game you can drop into your repo. * `README.md` snippet (with a play badge) that links to the GitHub Pages site where the game will be hosted. * Quick steps to enable GitHub Pages for your repo. --- ## index.html (single-file game) ```html Snake — Playable

Classic Snake — Play (Arrow keys / WASD)

Use arrows or WASD. Press Space to pause / resume.
Made for GitHub Pages
Score: 0
↑ / W→ / D↓ / S← / ASpace = pause
Tip: you can play on mobile using the on-screen buttons that appear when using touch.
Controls: Arrow keys / WASD. Refresh the page to restart if needed.
``` --- ## README snippet (use this in your repo's README.md) ```markdown # Play Snake in your browser [![Play Snake](https://img.shields.io/static/v1?label=Play&message=Snake&color=4c9aff&style=for-the-badge)](https://USERNAME.github.io/REPONAME) Click the badge above to open the game (hosted on GitHub Pages). > To host: add the provided `index.html` to your repository and follow the GitHub Pages steps below. ``` Replace `USERNAME` and `REPONAME` with your GitHub username and repository name. --- ## How to host on GitHub Pages (quick) 1. Put `index.html` at the repository root *or* inside a `docs/` folder. 2. Commit & push. 3. In GitHub, go to **Settings → Pages** for that repo. 4. Under **Build and deployment**, choose the branch and folder (`/ (root)` or `/docs`) and click **Save**. 5. After a minute the site will be available at `https://USERNAME.github.io/REPONAME`. > If you're using a custom domain, follow GitHub's instructions for DNS and the `CNAME` file. --- ## Optional: Animated GIF fallback for README If you prefer the game to *look* playable directly in the README, make a short screen-record GIF of the game and add it in README as `![Play demo](assets/snake-demo.gif)`.
